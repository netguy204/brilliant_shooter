Brilliant Shooter
-----------------

Shoot everything that isn't you. A particle system demo masquerading as a game.

<pre>
$ git clone https://github.com/netguy204/brilliant_shooter
$ cd mindless_shooter
$ git submodule init
$ git submodule update
$ make
$ ./run.sh
</pre>
